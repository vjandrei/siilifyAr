# siilifyAr

AR scanner for siilify image
https://ar-js-org.github.io/AR.js-Docs/

## Start

https://github.com/http-party/http-server#tlsssl
openssl req -newkey rsa:2048 -new -nodes -x509 -days 3650 -keyout key.pem -out cert.pem

## Run

npx http-server ./ -S
